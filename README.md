# budsUI

A comprehensive World of Warcraft user interface addon for WoW 3.3.5 (Wrath of the Lich King).

**Based on [KkthnxUI](https://github.com/kkthnx-wow/KkthnxUI) by Kkthnx**

## Features

- ✅ Full WoW 3.3.5 compatibility (Ascension.gg tested)
- 🎨 Custom color scheme (blue/green/red)
- 📦 Enhanced bags system
- 🎯 Custom unitframes
- ⚙️ Easy configuration GUI
- 🔧 Addon manager built-in

## Installation

1. Download the latest release
2. Extract both `budsUI` and `budsUI_Config` folders to your `World of Warcraft/Interface/AddOns/` directory
3. Restart WoW or type `/reload` in-game
4. Configure the addon using `/buds`

## Commands

- `/buds` - Open configuration GUI
- `/addons` - Open addon list
- `/uihelp` - Show help

## WoW 3.3.5 Compatibility

budsUI works perfectly on WoW 3.3.5 private servers like Ascension.gg.

All core features are fully functional with graceful degradation for missing API elements.

For technical details, see [budsUI/VERSION_COMPATIBILITY.md](budsUI/VERSION_COMPATIBILITY.md).
